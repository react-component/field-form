## API

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| visible | Whether the screen is visible | boolean | false |
| children | Content to be rendered | ReactNode | - |
| className | Additional class name | string | - |
| style | Additional style | CSSProperties | - |
